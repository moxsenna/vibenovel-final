# VibeNovel Agent Rule — UI/UX Product Rules

Default UX must be beginner-friendly.

Use plain-language labels:

- Story Bible → Fondasi Cerita
- Canonical Facts → Fakta yang Dikunci
- Reveal Gate → Jadwal Rahasia
- Beat Contract → Arahan Adegan
- Context Packet → Bahan Aman untuk AI
- Validator → Cek Otomatis
- Unlockability → Potensi Unlock

Simple by default, powerful by choice.

Do not expose advanced technical controls to beginner users unless they opt in.

Mobile readability matters. Writing and publishing pages must be comfortable on mobile.
