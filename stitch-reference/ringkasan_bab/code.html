Bab 4: Rahasia Hutan

Intisari Bab

Bab ini difokuskan pada pertemuan pertama Elara dengan penjaga hutan kuno. Ketegangan memuncak saat ia tidak sengaja memicu perangkap magis, memaksanya untuk menggunakan kemampuannya yang belum teruji. Pertemuan ini tidak hanya membuka fakta baru tentang asal-usul sihir di desa, tetapi juga memperdalam kecurigaan Kael terhadap motif tersembunyi tetua desa.
