# 15 — Publish Package and Growth Tools

## Purpose

Setelah bab selesai, VibeNovel membantu user menyiapkan bahan publish.

## Publish package contents

- judul bab,
- teaser,
- caption promosi,
- tags/genre,
- comment bait,
- next chapter teaser,
- mobile preview,
- copy buttons.

## KBM/mobile orientation

Copy harus cocok untuk pembaca serial mobile:

- hook cepat,
- bahasa jelas,
- tidak terlalu sastra,
- teaser memancing komentar,
- tidak spoiler berlebihan.

## MVP scope

Sprint 1: static/dummy UI.

Later MVP: generate publish package from accepted chapter summary and reader promise.
