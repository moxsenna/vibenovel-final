# 11 — UI/UX and User Modes

## UX principle

```txt
Simple by default, powerful by choice.
```

## Modes

### Beginner Mode

Default. Bahasa sederhana. Tidak ada istilah teknis. User diarahkan lewat chat, kartu, dan tombol aksi.

### Creator Mode

User bisa mengedit Fondasi Cerita, Outline, style, dan melihat warning lebih detail.

### Advanced Mode

Opt-in. User bisa inspect Context Packet, Reveal Schedule, Character Knowledge, validator report, dan version history.

## Main navigation

- Beranda

- Fondasi Cerita

- Ruang Tulis

## Acceptance criteria

- User awam tidak melihat kompleksitas teknis.
- Advanced controls tidak mengganggu default flow.
- Semua halaman utama punya empty/dummy state.
- Mobile layout bukan sekadar desktop diperkecil.
